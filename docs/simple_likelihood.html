How to use an external dataset?¶

Two options

  • From ini file with the existing cosmological functions

  • Without cosmological functions

From ini file with the existing cosmological functions¶

You can propose any function that you want. Define your parameters with Parameter

[custom]
...

model = LCDM

datasets = generic
path_to_data = /home/cosmocicatais/panth15.txt
path_to_cov = /home/cosmocicatais/panth15cov.txt
fn = distance_mod
...

Without cosmological functions¶

In this case, we need to make a Python script instead to use the ini file.

from simplemc.DriverMC import DriverMC
from simplemc.cosmo.Parameter import Parameter
